## How to run
#### Locally
1. Run `yarn install` in the root dir
2. Run `yarn chain` in one terminal
3. Once the first terminal is dumping ETH network logs, run `yarn deploy && yarn start` in the second terminal
#### Docker
1. See readme in `docker` dir
